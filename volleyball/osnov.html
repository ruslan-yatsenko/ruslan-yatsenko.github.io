Основи

Основні Правила Гри

  1. Склад команди: Кожна команда має шість гравців на майданчику. Усі гравці, крім ліберо, можуть атакувати та захищати.
  2. Подача: Гра починається з подачі. Гравець повинен вдарити м'яч через сітку, щоб м'яч потрапив у майданчик суперника.
  3. Розіграш м'яча: Кожна команда має право на три торкання, щоб повернути м'яч через сітку (не враховуючи блок).
  4. Атака та Захист: Гравці використовують різні техніки для атакуючих ударів, блоків і захисних дій. Основні удари — це напад, блок, підсічка, прийом і пас.

Основні техніки гри

  1. Подача: є кілька видів подач: верхня, нижня, планеруюча, силова.
  2. Прийом: техніка прийому подачі, щоб підготувати м'яч для атаки.
  3. Пас: передача м'яча для підготовки атаки.
  4. Атака: потужний удар по м'ячу, спрямований у майданчик суперника.
  5. Блокування: захисний маневр для зупинки атаки суперника.
